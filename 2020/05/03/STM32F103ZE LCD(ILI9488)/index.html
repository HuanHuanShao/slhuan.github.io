#1 环境

  • 硬件：STM32F103ZE开发板、3.4寸ILI9488驱动的LCD和8002A触摸芯片集成的显示器
  • 软件：Keil MDK V5.32
    #2 ILI9488基础知识
    ##2.1 简介
      ILI9488是一个320（RGB）×480点的可变显示驱动芯片，频率是16.7Mhz。其包含了960路原驱动器，480路门驱动器，345600字节的GRAM和供电电路。
    ##2.2 系统接口
      ILI9488的系统接口由其三个引脚IM[2:0]的高低点评决定，如下表：



      ILI9488支持MIPI-DBI Type B接口，通过命令控制IR寄存器和CR寄存器。它也有以恶18-bit的写数据寄存器WDR和读数据寄存器RDR。MPU发送数据给WDR，然后再通过芯片的内部操作保存到GRAM；MPU可以直接从GRAM读取数据。读写的操作电平如下表：



      通过表格分析，我们首先要知道我们选择的LCD显示器使用的哪一种接口，通过显示器资料可以找到，例如我们所选择的在文档中就提及到其接口类型是MCU 16bit接口，即使用的Type B接口。

#3 FSMC
  LCD显示器的片选CS连接的是STM32F103的FSMC_NE4，对应的FSMC存储区的BANK4，地址线RS连接的是FSMC_A23，所以在单片机内映射的首地址是（0x6C00 0000 + (1<<(23+1))-2）；
